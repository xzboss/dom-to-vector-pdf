# dom-to-vector-pdf

DOM 转矢量 PDF

1. 克隆元素时对原页面的影响
   - 内联控制（current）
     - 样式优先级
   - Shadow DOM
   - iframe
2. iconfont 大小问题：目前是默认 16px 字体大小来做的缩放
3. 原来的 DOM 中本身存在的 SVG 标签是用的内联样式来决定的样式
   - 目前只支持内联中属性名和元素属性名一致的情况
4. 页面整体转换出来的文字对比背景要偏下一点
   - 目前是暂时所有文字整体上移 3 个像素单位
5. 不支持项

- 目前不支持导出图片背景
- 目前不支持 canvas 导出

6. 字体

- 目前只支持一种，注册字体时 fontId 必须一样
- 多种字体考虑兼容 woff2 格式

7. 图片

- 图片导出效果不理想
